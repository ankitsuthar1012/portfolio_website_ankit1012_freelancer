# Portfolio Website - Web Developer & AI Solutions Expert

A high-converting, motion-rich portfolio website built to attract freelance clients. Combines modern web development with AI-powered solutions showcase.

## 🚀 Quick Start

### 1. Install Dependencies
```bash
npm install
```

### 2. Configure Environment
```bash
cp .env.example .env
# Edit .env with your settings
```

### 3. Run the Server
```bash
# Development (with auto-reload)
npm run dev

# Production
npm start
```

### 4. Open in Browser
Visit `http://localhost:3000`

## ✨ Features

### Frontend
- **Hero Section** - Typing animation, particles, parallax scroll
- **About Section** - Professional bio with highlights
- **Skills** - Tech stack organized by category
- **Projects** - 6 portfolio projects with 3D tilt effect
- **Services** - 4 freelance services with pricing
- **Testimonials** - Client reviews with hover effects
- **Contact** - Form with validation + backend integration

### Motion & Animations
- Scroll-triggered reveal animations (Intersection Observer)
- 3D card tilt effect on project cards
- Floating particles in hero section
- Magnetic button hover effect
- Parallax scrolling on hero
- Cursor glow follow effect
- Typing text rotation
- Animated counters
- Scroll progress bar
- Navbar hide/show on scroll

### Backend
- Express.js REST API
- Contact form with validation & sanitization
- Rate limiting (10 requests/15 min)
- Helmet security headers
- Optional email notifications via SMTP
- Admin endpoint to view messages
